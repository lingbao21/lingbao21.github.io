2.2 注释

注释的目的是对程序进行说明，不会被执行

  • 单行注释：# 注释内容
  • 多行注释："""注释内容"""
"""
这是字面量的输出方法
这是注释的内容
"""
print(123) # 这是int
print(13.14) # 这是float
print("大帅比绳军") # 这是string

思考：

"""
这个程序的输出内容是什么
"""
print("绳军") # print("大帅逼")

2.3 变量

变量就是程序在使用时，记录数据用的

变量的格式为：变量名 = 变量值

变量的特征：变量的值是可以改变的

"""
演示：
变量的用法：
"""
# 定义一个变量来表示钱包的余额
money = 50
# 利用print语句，输出钱包余额
print("钱包原本余额为：",money,"元")
# 买雪糕花了十元，计算剩下的余额
money = money  - 10
# 输出剩下的钱
print("买雪糕花了10元，钱包余额为：",money,"元")

补充内容

  • 变量可以是int,float,str等，在输出变量时，不需要加双引号强调是字符串
  • 定义变量为字符串时，字符串要加引号

  • print如何输出多份内容：

    print(内容1,内容2,内容3, ... )

  • python的简单运算

    加（+），减（-），乘（*），除（/）

练习题

"""
练习题
"""
# 定义一个变量：钱包余额money，初始值为50元
money = 50
print("钱包初始余额为：",money,"元")
# 买雪糕花费：10元，买巧克力花费：5元
money = money - 10
print("买雪糕花费：10元")
money = money - 5
print("买巧克力花费：5元")
# 输出钱包最终余额： 元
print("钱包最终余额为",money,"元")

2.4 数据类型

初学者常见的数据类型

  • 整型int
  • 浮点型float
  • 字符串

利用type()来查看数据类型，既可以查看字面量的数据类型，又可以查看变量的数据类型

"""
数据类型
利用tyoe()查看数据类型
"""
#查看字面量的数据类型
# 方式一：
print(type(123))
print(type(123.456))
print(type("我是大帅逼"))

# 方式二：
int_type = type(123)
print(int_type)
float_type = type(123.456)
print(float_type)
str_type = type("我是大帅比")
print(str_type)


#查看变量的数据类型
# 方式一：
int_type = 123
print(type(int_type))

# 方式二：
f = 123.456
float_type = type(f)
print(float_type)

2.5 数据类型转换

数据类型之间，在特定的场景之间是可以相互转换的，如数字与字符串之间

数据转换用途广泛且有必要

函数(x) 说明
int(x) 将x转换成整数
float(x) 将x转换成浮点数
str(x) 将x转换成字符串 
"""
字符串转换
"""

# 整数转字符串
num1 = str(1234)
print(type(num1),num1)

# 浮点转字符串
print(type(str(213.456)),str(213.456)) # 通过中间变量的方式，可以使程序更简洁

# 字符串转整型
num2 = int("1234")
print(type(num2),num2)

"""
错误示例：
num3 = int("大帅比")
print(type(num3),num3)
"""

# 整型转浮点型
num4 = float(1234)
print(type(num4),num4)

# 浮点型转整型
num5 = int(123.456)
print(type(num5),num5) # 浮点型转整型会失去精度，即抹掉小数点后面的数，不是四舍五入

2.6 标识符

标识符就是用户在编程时所使用的一系列的名字，用于给变量，类，方法等命名

  • 标识符只允许出现英文，中文（不推荐），数字，下划线四类元素，其他内容均不被允许

  • 数字不能作为开头

  • 对大小写敏感

  • 不能使用关键字

变量的命名规范：

  • 见名知义
  • 下划线命名法
  • 英文字母全小写
